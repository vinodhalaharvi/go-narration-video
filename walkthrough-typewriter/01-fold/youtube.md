#Shorts

The shape of your program should mirror the shape of your data.

In this short I build a generic `Fold[A]` function over a recursive tree, then compose it with predicates to express queries that read like English. No builder pattern, no interfaces — just functions and Go generics.

The pattern works for any recursive structure: file systems, JSON, ASTs, dependency graphs.
